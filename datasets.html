Datasets

Details on the Training, Validation, and Final Challenge Datasets coming soon!

Training datasets

These are labeled datasets that allow the proponents to train their algorithms. The goal here is to provide some comfort with the dataset format and how to process the information.

Validation/evaluation datasets

We provide a private validation dataset to allow the participants to test their model for submission. The evaluation metrics match those of the overall evalution for the test datasets.

Final Challenge datasets

Performance on the Final challenge datasets will determine challenge scores. Please see the individual challenges for more details on the evaluation. Submission scoring code is provided on each challenge's GitHub repository as well.
